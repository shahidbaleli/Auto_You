Code of Conduct

Be respectful and helpful. We're building this tool for creators and hobbyists — please keep interactions constructive.

- Treat everyone with respect.
- Be kind in feedback and reviews.
- No harassment, abuse, or hate speech.
- If you see a problem, report it in an issue or contact the maintainers.

The maintainers reserve the right to remove comments or contributions that violate these guidelines.

If you need to contact the maintainers privately, open an issue marked "private" and include the reason for privacy.
